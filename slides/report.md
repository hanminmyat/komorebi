## Notes

Komorebi (木漏れ日 — sunlight filtering through leaves) is an AI-assisted family memory platform. MVP scope includes creating audio, photo, and mixed memory capsules, uploading/recording media, organizing albums with ordering, and sharing public capsule links — built with Next.js (App Router), TypeScript, Tailwind CSS, and Supabase (Auth + DB + Storage).
